# Compose Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 为 `@g-motion/animation` 增加纯数据 `compose()` 模板，并让 `timeline()` 在 authoring 阶段展开 composition layer。

**Architecture:** `compose()` 只负责产出不可变模板对象，不参与 runtime。`timeline()` 在创建 authoring model 时将 composition layer 转换成普通 layer，再复用现有 binding、controller、state snapshot 和 track 编辑链路。

**Tech Stack:** TypeScript, Vitest, workspace package `@g-motion/animation`
