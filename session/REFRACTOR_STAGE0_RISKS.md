# 阶段 0 风险清单

## 技术风险
- **基准脚本覆盖不足**：关键 API 路径缺少直接基准测试，导致基线失真  
  影响：高｜缓解：新增核心 API 基准用例与覆盖清单
- **WebGPU 运行环境缺失**：本地或 CI 无法提供 WebGPU  
  影响：高｜缓解：建立 GPU 运行环境清单与条件执行策略
- **长基准运行时间**：核心基准耗时过长影响可重复性  
  影响：中｜缓解：拆分为核心/扩展两层基准并分批执行

## 兼容性风险
- **基线依赖 Node/浏览器版本差异**：性能指标受运行时影响  
  影响：中｜缓解：锁定 Node 版本与硬件配置记录

## 性能风险
- **基线采集指标不一致**：不同测试输出维度不统一  
  影响：高｜缓解：统一指标定义与输出格式
- **GPU 数据传输路径缺少监控**：读回与缓冲池指标缺失  
  影响：高｜缓解：补充 GPU buffer/dispatch 统计并记录

## 流程风险
- **变更提案未落地执行**：团队未强制遵守治理流程  
  影响：高｜缓解：在提交流程中要求 OpenSpec 变更项
